# Rock-Paper-Scissors
A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Features

- **Play Rock, Paper, or Scissors against the computer**
- **Simple and intuitive interface**
- **Instant feedback on game results**
